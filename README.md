# ⚒️ SkillForge — Платформа для изучения программирования

Интерактивная платформа для обучения программированию. Изучайте темы, решайте задачи и отслеживайте прогресс.

---

## 🌐 Демо

🔗 **Ссылка на сайт:** [https://mikaaa5z.github.io/SkillForge/](https://mikaaa5z.github.io/SkillForge/)

---

## 📚 О проекте

**SkillForge** — это веб-платформа, где можно:

- 📖 Изучать темы программирования (Python, C++, Pascal, JavaScript)
- 💻 Решать интерактивные задачи с проверкой ответов
- 📊 Отслеживать свой прогресс и статистику
- 👤 Создавать личный кабинет с аватаром
- 🏆 Получать статистику решённых задач

---

## 🛠️ Технологии

| Технология | Назначение |
|------------|------------|
| **HTML5** | Структура страниц |
| **CSS3** | Стили и адаптивность |
| **JavaScript** | Вся логика и взаимодействие |
| **LocalStorage** | Сохранение прогресса и аватара |
| **Supabase** | База данных (регистрация) |
| **Node.js + Express** | Бэкенд-сервер |
| **GitHub Pages** | Хостинг фронтенда |
